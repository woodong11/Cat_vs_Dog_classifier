# Cat_vs_Dog_classifier
캐글 cat and dog 데이터셋으로 만든 개 고양이 분류기입니다.
'길벗의 딥러닝 파이토치 교과서'를 참고했습니다.

사용한 데이터셋 링크: https://www.kaggle.com/datasets/tongpython/cat-and-dog
